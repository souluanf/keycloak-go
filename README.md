# Autenticação com KeyCloak


### Framework e Bibliotecas
- **GO 1.17**: Develop


### Execução

1. Acessar pasta do código:
    ```shell
    cd goglient
    ```

2. Instalar dependências
   ```shell
   go get 
   ```

3. Rodar projeto
    ````shell
    CLIENT_ID=client CLIENT_SECRET=secret_key go run main.go
    ````
